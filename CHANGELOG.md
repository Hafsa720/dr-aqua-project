# RapidBizz Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Complete business website template with Home, Services, Projects, Team, Careers, and Contact pages
- Professional design system with consistent branding
- Framer Motion animations throughout the application
- Enhanced GitHub automation workflows
- Comprehensive Open Graph meta tag generation
- Advanced sitemap configuration with custom priorities
- Project-specific VS Code snippets
- OKLCH color system implementation

### Changed
- Updated to Next.js 15 with App Router
- Migrated to React 19
- Enhanced TypeScript configuration
- Improved component structure and organization
- Updated README with comprehensive documentation

### Removed
- Original starter template branding
- Outdated dependencies and configurations

## [1.0.0] - 2024-01-01

### Added
- Initial project setup with Next.js, TypeScript, and Tailwind CSS
- Core component library (buttons, links, forms)
- Basic GitHub workflows for CI/CD
- Development tooling (ESLint, Prettier, Husky)
- Jest testing configuration
