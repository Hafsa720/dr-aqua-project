# Project Hero Images

This directory contains hero images for project case studies.

## Required Images:

- `medconnect-hero.jpg` - Healthcare platform hero image
- `finflow-hero.jpg` - FinTech banking solution hero image
- `shopflow-hero.jpg` - E-commerce platform hero image
- `edutech-hero.jpg` - Online learning platform hero image
- `smartcity-hero.jpg` - Smart city dashboard hero image
- `logistics-hero.jpg` - Supply chain management hero image
- `default-hero.jpg` - Default fallback hero image

## Recommended Specifications:

- Format: JPG or WebP
- Dimensions: 1200x630px (optimal for social sharing)
- File size: < 500KB
- Content: Professional, relevant to the project industry
- Quality: High resolution for crisp display on all devices

## Alternative Images:

Each project should also have:

- `{project}-dashboard.jpg` - Dashboard/interface screenshot
- `{project}-mobile.jpg` - Mobile app interface (if applicable)

These images are referenced in the markdown frontmatter of each case study file in `/src/content/case-studies/en/`.
