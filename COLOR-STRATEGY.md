# Color Strategy - 70/30 Distribution

## Brand Colors

### Primary Color (70-80% usage)

- **Color**: #082856 (Deep Blue)
- **OKLCH**: oklch(0.19 0.09 258.4)
- **Usage**: Main brand color, backgrounds, borders, text, primary UI elements

### Secondary Color (20-30% usage)

- **Color**: #FF8621 (Orange)
- **OKLCH**: oklch(0.67 0.27 68.3)
- **Usage**: CTAs, highlights, important actions, accents, emphasis

## Color Distribution Strategy

### Navigation & Header (Primary Dominant)

- Background: White/Primary-50
- Text: Primary-900
- Links: Primary-700 (hover: Primary-500)
- CTA Button: Secondary-600 (hover: Secondary-700)
- Cart Badge: Secondary-600

### Hero Sections (70/30 Split)

- Background: Primary-50 to Primary-100 gradients
- Headings: Primary-900
- Body text: Primary-700
- Badges: Primary-500 or Secondary-500 (for emphasis)
- Primary CTA: Secondary-600
- Secondary CTA: Primary outline

### Content Sections (Primary Dominant)

- Backgrounds: White, Primary-50
- Cards: White with Primary-200 borders
- Text: Primary-900 (headings), Primary-700 (body)
- Icons: Primary-500 or Secondary-500 (for important items)

### Product Cards (Strategic Color Use)

- Border: Primary-200 (hover: Primary-300)
- Category badge: Primary-600
- Discount badge: Secondary-600
- Price: Primary-700
- Rating stars: Yellow-400 (exception for UX clarity)
- "Add to Cart" button: Primary-600 (main action)
- "View Details" button: Primary outline

### Buttons (Clear Hierarchy)

1. **Primary Action**: Secondary-600 bg (hover: Secondary-700)
2. **Secondary Action**: Primary-600 bg (hover: Primary-700)
3. **Tertiary Action**: Primary-600 outline (hover: Primary-50 bg)
4. **Ghost Action**: Transparent (hover: Primary-50)

### Forms & Inputs (Primary System)

- Border: Primary-300
- Focus: Primary-500 border + Primary-200 ring
- Label: Primary-900
- Helper text: Primary-600
- Error: Red (exception for UX clarity)

### Footer (Primary Dominant)

- Background: Primary-900
- Text: Primary-100
- Links: Primary-200 (hover: Secondary-400)
- Headings: White
- Social icons: Primary-300 (hover: Secondary-500)

## Color Usage Rules

### DO ✅

- Use primary colors for 70-80% of visual elements
- Use secondary color for important CTAs and accents (20-30%)
- Maintain consistent color hierarchy
- Use lighter primary shades for backgrounds
- Use darker primary shades for text
- Reserve secondary color for user actions and highlights

### DON'T ❌

- Don't use secondary color as background (except small badges)
- Don't mix primary and secondary equally
- Don't use too many color shades in one component
- Don't use secondary for body text
- Don't overuse accent colors

## Component-Specific Guidelines

### Cards

- Background: White
- Border: Primary-200
- Hover border: Primary-300
- Title: Primary-900
- Description: Primary-700
- Primary action: Secondary-600
- Secondary action: Primary outline

### Badges

- Category/Info: Primary-600 bg
- Discount/Sale: Secondary-600 bg
- Status: Green/Yellow (exceptions)
- Size: Small, minimal

### Links

- Default: Primary-700
- Hover: Primary-500
- Visited: Primary-800
- Active: Secondary-600 (in navigation only)

### Shadows & Borders

- Default border: Primary-200
- Hover border: Primary-300
- Focus ring: Primary-200/20
- Shadow: Primary-900/5 to 10

## Page-Specific Distribution

### Home Page

- Hero: 60% Primary, 40% White, CTA: Secondary
- Features: 70% Primary, 30% White
- Products: 75% Primary, 25% White/Secondary accents
- Services: 80% Primary, 20% White
- Footer: 90% Primary, 10% Secondary accents

### Shop Page

- Filters: 100% Primary system
- Product Grid: 70% Primary, 30% White
- CTAs: Secondary-600 (Add to Cart only)
- Navigation: Primary outline buttons

### Product Detail

- Product info: 70% Primary for structure
- Actions: Secondary for "Add to Cart"
- Tabs: Primary-600 active state
- Specifications: Primary-100 backgrounds
